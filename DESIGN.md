# Design System ‚ÄĒ lorenzo.studio

> Read this before any visual or UI change. Fonts, colours, spacing, and
> aesthetic direction are defined here. Don't deviate without explicit approval.
> Tokens live in [`src/app/globals.css`](src/app/globals.css) (`:root`).

## Product Context
- **What this is:** a freelance studio-of-one portfolio that sells outcomes ‚ÄĒ websites & e-commerce, online visibility, social content, automation of repetitive work, web-data collection, and AI agents for solo founders.
- **Who it's for:** small and large businesses in the **Italian market** (IT primary; EN/SV also supported).
- **Memorable thing:** *real craft, not a template.* Every choice serves the impression that a human with taste built this ‚ÄĒ which is what converts skeptical clients into consulting calls.
- **Project type:** marketing portfolio + funnel (content ‚Üí membership ‚Üí consulting), with an in-app AI agent layer.

## Aesthetic Direction
- **Direction:** warm neo-brutalist. Exposed structure (thick ink borders, hard offset shadows), warm paper canvas, mono technical labels, expressive serif display.
- **Decoration level:** intentional ‚ÄĒ borders + hard shadows + light paper texture do the work; no blur, no gradients, no glassmorphism.
- **Mood:** hand-built, confident, growth-minded. Craft over polish-for-its-own-sake.
- **Anti-slop guardrails:** no purple gradients, no 3-column icon-in-circle grids, no centered-everything, no soft drop shadows, no Inter/Geist/system-ui as display or body.

## Typography
- **Display / Hero:** **Fraunces** (variable "soft serif", optical sizing). Used on hero h1, `.section-head__title`, project card titles, detail `<h1>`. Loaded via `next/font` ‚Üí CSS var `--font-fraunces`, exposed as `--font-display`.
- **Body / UI:** **General Sans** (Fontshare). Loaded via `<link>` in [`layout.tsx`](src/app/layout.tsx); exposed as `--font-sans` / `--font-ui`.
- **Mono / Labels:** **JetBrains Mono** (`next/font`) ‚Üí `--font-mono`. Eyebrows, stats, badges, code, the wordmark accent context.
- **Scale (px):** display `clamp(34, 5.4vw, 96)` ¬∑ h1 28 ¬∑ h2 22 ¬∑ lg 18 ¬∑ base 16 ¬∑ sm 14 ¬∑ xs 12 ¬∑ badge 10.
- **Weights:** 400 / 500 / 600 (display headings 500‚Äď600; never 700+ on Fraunces display).
- **Tracking:** display `-0.02em`; tight `-0.02em`; never tighter than `-0.02em` on the serif (it cramps).

## Color
- **Approach:** restrained ‚ÄĒ the **Parchment & Forest** palette: a light parchment canvas, ink text, an ochre accent for actions, deep forest for links, with teal/sage as cool accents. Colour is meaningful, not decorative.
- **Palette source (7):** parchment `#F5EEDF` ¬∑ ink `#221E17` ¬∑ forest `#143A2B` ¬∑ ochre `#E8A12C` ¬∑ coral `#E5623F` ¬∑ teal `#3E867A` ¬∑ sage `#9DB57F`.
- **Canvas:** page ‚ÄĒ **parchment** `#F5EEDF` ¬∑ panel-yellow ‚ÄĒ warm sand `#EFE6D1` ¬∑ panel-grey ‚ÄĒ `#F7F1E3`. `--canvas-grad` is a whisper-soft diagonal parchment gradient (no radial "studio pool" ‚ÄĒ fixed-attachment made it band across every screen). The hero is now a separate full-bleed cinematic stage (`HeroMotion`) above the parchment page, not blended into the canvas.
- **Primary accent ‚ÄĒ ochre (gold).** Token names stay `*-green*` so every reference cascades unchanged. Ochre is light, so fills take **dark ink** text, never white.
  - `--accent-green` `#E8A12C` (ochre) ‚ÄĒ fills (badges, section-number chips, hero highlight, pillar index, primary CTA) with **dark `--btn-ink`** text.
  - `--accent-green-deep` `#143A2B` (forest) ‚ÄĒ text-on-light (links, wordmark accent, value labels); a rich green, high contrast on parchment.
  - `--accent-green-bright` `#F2B84E` (bright ochre) ‚ÄĒ dark-mode colored shadow + dark-mode price text.
  - Legacy `--accent-coral*` tokens are **aliases** of the ochre accent above.
- **Text-on-accent:** `--fg-on-coral` / `--accentInk` resolve to `--btn-ink` `#221E17` (fixed; does **not** flip in dark, because the ochre fill is light in both themes). White-on-accent is retired.
- **Warm secondary:** peach ‚Üí **sage** `#9DB57F` / peach-deep ‚Üí deep sage `#88A268` ‚ÄĒ `.neo-btn` background (friendly secondary CTA, dark text). Sage is deliberately given presence here.
- **Decor:** **teal** `#3E867A` ‚ÄĒ cold accent + "available" status dot. (The FREE tier badge sits on teal with light text.)
- **Coral** `#E5623F` is the palette's reserved warm pop ‚ÄĒ available for emphasis, not wired into the core tokens yet.
- **Ink:** border/body/shadow ‚ÄĒ **ink** `#221E17` ¬∑ muted `#6B6253`.
- **Semantic error:** `--error` `#C0392B` (dark-mode `#FF7A6E`). Use `--error` only for error/danger states (form validation, never decoration). Coral is a brand accent, not the error colour.
- **Dark mode:** warm-charcoal canvas matched to the dark hero frames ‚ÄĒ base `#29251f`, gradient `#322e28‚Üí#1c1916` with a soft light pool where the hero sits (mirrors the light `--canvas-grad`). Panels `#332e27` / `#2c2821` (raised), ink `#F4F1EA`, colored shadow `#E8A12C` (ochre), text/link accent lifted to light ochre `#E8B25E` (forest is invisible on charcoal), input surface `#221f1a`. Ochre fills keep dark text in both themes; canvas/grad stay frame-matched.

## Spacing
- **Base unit:** 4px. **Density:** comfortable.
- **Scale:** 4 ¬∑ 8 ¬∑ 12 ¬∑ 16 ¬∑ 20 ¬∑ 24 ¬∑ 32 ¬∑ 40 ¬∑ 48 ¬∑ 64 (`--space-1`‚Ä¶`--space-16`).

## Layout
- **Approach:** hybrid ‚ÄĒ disciplined grid for cards/app, editorial for the hero.
- **Max content width:** 1200px (`--container-max`); agent pages 1440.
- **Card grid:** `repeat(auto-fit, minmax(280‚Äď320px, 1fr))`, gap 24‚Äď32.
- **Border radius:** sm 2 ¬∑ base 4 ¬∑ lg 12 ¬∑ full 9999. Cards/panels use `lg`; controls use `base`.
- **Project cards:** one shape everywhere ‚ÄĒ `.neo-card` body, 3px ink border, 16/10 cover, hard offset shadow, hover lift `translate(-2px,-2px)`, serif title, mono `Valore` line.

## Motion
- **Approach:** intentional. Entrance reveals + meaningful hover/state transitions; nothing decorative.
- **Easing:** `--ease` `cubic-bezier(0.16, 1, 0.3, 1)`.
- **Duration:** fast 150ms ¬∑ base 200ms ¬∑ slow 300ms.
- **Signature:** neo offset shadows shift on hover (panel 8px, card 6px, control 4px); buttons press to `translate(6px,6px)` with shadow removed.
- **Premium layer (homepage):** Framer Motion staggered reveals on card grids (`Reveal.tsx` ‚Üí `RevealStagger`/`RevealItem`), magnetic hero CTAs (`MagneticButton.tsx`), count-up hero stats (`CountUp.tsx`), a single diagonal card light-sweep on hover (`.neo-card--shine`), growing emerald accent rail (`.accent-rail`), and a pause-on-hover edge-faded marquee (`.marquee`). All honor `prefers-reduced-motion`. NB: Framer `whileInView` is used only on non-pinned sections (Services/EntryGrid/Plans); the Hero `#top` pin and the Work sticky gallery must never be wrapped in a transformed motion container.

## Premium Elevation (the "expensive" finish ‚ÄĒ keeps the system above intact)
- **Goal:** make the warm neo-brutalist system read high-end without abandoning the accent / Fraunces / paper. Added depth + sheen + texture on top of the existing tokens; nothing was replaced.
- **Grain:** `--grain` baked low-alpha SVG noise, composited as the top background layer of `body` for tactile paper.
- **Sheen:** `--accent-green-sheen` (in-hue 145¬į ochre/gold) on primary CTAs (`.neo-btn--primary`), section-number chips, hero highlight, scroll progress. Disciplined depth, not a decorative rainbow.
- **Shadow depth:** `--shadow-card` / `--shadow-card-hover` / `--shadow-panel` keep the hard ink offset and add a faint ambient pool so panels lift off the paper.
